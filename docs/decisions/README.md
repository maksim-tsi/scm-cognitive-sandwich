# Decision Records

Architecture Decision Records (ADRs) for major technical choices.

## Files

- `001-llm-models-selection.md`: accepted model pinning for Gemini, Mistral, and Groq.
- `002-otel-phoenix-project-attribution.md`: accepted attribution normalization strategy for Phoenix project routing.

## Usage

When a durable architecture choice changes, add a new ADR file with:

1. Status
2. Context
3. Decision
4. Consequences
